# talisman
sample:
```js
let code = talisman(
  '02', // poison skill one
  '02', // sleep skill two
  10, // skill point one
  10, // skill point two
  1, // has 1 slot
  2, // rare-2 (purple)
);
console.log(code);

```
output:
```

_L 0x2174FE14 0xA2816501
```
